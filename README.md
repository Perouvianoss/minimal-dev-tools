# Minimal Dev Tools

Minimal Dev Tools is a lightweight collection of essential tools designed to streamline development workflows.

## Features

- Lightweight and easy to use.
- Focused on simplicity and productivity.
- Includes only the most essential tools.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/minimal-dev-tools.git
   ```
2. Navigate to the project directory:
   ```bash
   cd minimal-dev-tools
   ```
